* if using npm *

Build front-end for "reset-password"

cd frontend,
npm install,
npm run build,
cd ..

Rename .env.local ke .env,
npm install,
npm run dev,


* if using pnpm *

npm install -g pnpm,

Build front-end for "reset-password"

cd frontend,
pnpm i,
pnpm build,
cd ..,

Rename .env.local ke .env,
pnpm i,
pnpm dev,
